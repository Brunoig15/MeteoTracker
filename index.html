Meteo Tracker

Introducción

MeteoTracker es una estación meteorológica que está pensada para ser lo más económica y portátil posible, utilizando sensores y un Arduino como microcontrolador. Además, garantiza el fácil acceso de los datos del clima a los ciudadanos a través de esta página web en la que todos pueden acceder gratuitamente.

Objetivos

  • Construir una estación meteorológica económica y portátil.
  • Proporcionar acceso gratuito a los datos climáticos a la población local a traves de una pagina.
  • Promover la conciencia del cambio climático y comprender el clima local.

Proyecto cooperativo

Meteo Tracker se vincula con el proyecto "LoboSat" para obtener datos de variables climáticas a alturas superiores. Los datos recopilados por ambas estaciones estarán disponibles en una página web para que la población local pueda acceder a ellos.

Componentes

  • Placa Arduino: La placa Arduino, como el modelo Arduino Uno, es el cerebro de la estación meteorológica y se encarga de controlar y coordinar todos los componentes.
  • Sensor DHT22: Permite medir la temperatura y la humedad del ambiente.
  • Sensor BMP180: Es capaz de medir la presión atmosférica, lo que puede proporcionar datos para el pronóstico del tiempo.
  • Anemómetro: Un anemómetro mide la velocidad del viento. (En proceso de construccion)
  • Pantalla LCD: Muestra los datos recopilados, de temperatura humedad, presión, altura(a traves de un calculo con la presión) y en un futuro la velocidad del viento
